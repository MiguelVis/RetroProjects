INTRODUCTION
------------

Ummo is a graphical game for the Amstrad PCW.

1st of June, 1967.

A UFO from planet UMMO, was seen flying over San Jose de Valderas, Madrid, Spain.

Press said that... bla, bla, bla...

Look. This is a shoot'em up game, you know perfectly how it works, then...

Shoot the aliens or die!


GAME FILES
----------

UMMO.COM -- Command to run the game in CP/M.
UMMO.TXT -- This help file.


HOW TO PLAY
-----------

UP     : MOVE UP
DOWN   : MOVE DOWN
LEFT   : MOVE LEFT
RIGHT  : MOVE RIGHT
SPACE  : LASER
RETURN : BOMB
STOP   : PAUSE
EXIT   : ABORT GAME


TECHNICAL NOTES
---------------

This program was developed using MESCC (Mike's Enhanced Small C Compiler - my
own version of Small C), and some SAMARUC libraries (my GUI / Graphical User
